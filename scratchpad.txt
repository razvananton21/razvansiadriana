# Deployment Options

## Fastest Options (in order of simplicity)

1. **Vercel** - Connect GitHub repo or use `vercel` CLI (requires Node.js 18+)
   - Best option for Next.js projects
   - Automatic deployments, previews, analytics
   - Free tier available

2. **Netlify** - Connect GitHub repo or use drag-and-drop
   - Simple UI, automatic deployments
   - Free tier available
   - Great for static sites

3. **GitHub Pages** - Use the included GitHub Actions workflow
   - Free for public repositories
   - Automatic deployments via GitHub Actions
   - Available at username.github.io/repo-name

4. **Firebase Hosting** - Fast CDN, easy setup
   - Free tier available
   - Global CDN
   - Easy CLI deployment

5. **Surge.sh** - Super simple deployment
   - Free tier available
   - One command deployment
   - Custom domains on free tier

## Local Deployment Script
- Use `./deploy-local.sh` for manual GitHub Pages deployment
